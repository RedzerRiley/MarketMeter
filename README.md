# Git commands/instructions
To fully utilize Git use the commands below

If you do not have Git please download Git
https://git-scm.com/install/windows

and setup your Git in your terminal inputing your Github credentials
https://www.youtube.com/watch?v=wDRoduig_98

## Clone the repo to your terminal in your preferred IDE (ex: VSCode) 

``` bash
git clone https://github.com/RedzerRiley/MarketMeter

```

## Make sure your in the right branch
Example: 
``` bash
git switch your-branch
```
Make sure your in the right branch
``` bash
git branch
```
the * indicates what branch you are on

## Merging branches

If your code is dependent on someone else's code then pull their branch to yours
``` bash
git pull origin branch-name
```

## Pushing code to your branch
```bash
git add .
git commit -m "Your message"
git push
```
